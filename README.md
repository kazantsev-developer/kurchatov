# Проект: Эволюция разума — от алгебры до ChatGPT

Сделали совместно с сестрой Яной (ученицей 7 «А» Курчатовского класса) интерактивный таймлайн по физике, математике и информатике.
Это не просто слайды, а веб-приложение на стыке науки и IT.

## О проекте

- **Тема:** История развития математики, логики, вычислительной техники и искусственного интеллекта.
- **Цель:** Создать интерактивную презентацию вместо скучного PowerPoint.
- **Класс:** 7 «А» Курчатовский класс.
- **Предметы:** физика + математика + информатика.

## Технологии

- **React 19** — библиотека для UI.
- **Vite** — сборка проекта.
- **Tailwind CSS 4** — стилизация.
- **Framer Motion** — плавные анимации карточек.
- **HeroUI (NextUI)** — компоненты (Navbar, Card, Chip, Button).
- **Lucide React** — иконки.

## Архитектура проекта

Проект построен на **FSD (Feature Sliced Design)** современном подходе к организации кода. Это позволяет легко масштабировать проект, переиспользовать код и поддерживать чистоту архитектуры.

## Фишка

Вместо обычного PowerPoint сделали **интерактивное веб-приложение**:

- Плавная 3D-карусель с карточками.
- Адаптивный дизайн (работает на телефонах, планшетах и компьютерах).
- Картинки и информация об этапах эволюции разума.
- Индикаторы слайдов с возможностью переключения.
- Прогресс-бар в навбаре и кнопка «Старт» на титульном слайде.

Для школьницы — это самый топ!

## Как запустить

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
