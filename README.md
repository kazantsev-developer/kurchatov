# Проект: Биологическая очистка воды

Сделали совместно с сестрой Яной (ученицей 7 «А» Курчатовского класса) интерактивную презентацию по биологии + информатике.
Тут не просто слайды, а веб приложение на стыке биологии, экологии и IT.
Яна потихоньку вливается в программирование.

## О проекте

- **Тема:** Биологическая очистка воды (водоросли, бактерии, моллюски, биоплато)
- **Цель:** Создать интерактивную презентацию вместо скучного PowerPoint
- **Класс:** 7 «А» Курчатовский класс
- **Предмет:** Биология

## Технологии

- **React 19** библиотека для UI
- **Vite** сборка проекта
- **Tailwind CSS 4** стилизация
- **Framer Motion** плавные анимации карточек
- **HeroUI (NextUI)** компоненты (Navbar, Card, Chip, Button)
- **Lucide React** иконки

## Архитектура проекта

Проект построен на **FSD (Feature Sliced Design)** современном подходе к организации кода. Такой подход позволяет легко масштабировать проект, переиспользовать код и поддерживать чистоту архитектуры.

## Фишка

Вместо обычного PowerPoint сделали **интерактивное веб приложение**:

- Плавная 3D карусель с карточками
- Адаптивный дизайн (работает на телефонах, планшетах и компьютерах)
- Картинки и информация по биологической очистке воды
- Индикаторы слайдов с возможностью переключения

Для школьницы, я считаю, что это самый топ!

## Как запустить

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
