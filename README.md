# Проект: Очистка воды (7 класс)

Сделали совместно с сестрой Яной (ученицей 7 «А» Курчатовского класса) годную презентацию по биологии+информатике.
Тут не просто слайды, а интерактив на стыке экологии и IT.
Она потихонько вливается в программирование.

### Суть

- **Тема:** Технологии очистки воды
- **Стек:** React, Framer Motion (для плавных карточек), Tailwind/HeroUI.

### Фишка

Вместо обычного PowerPoint запилили веб приложение с анимациями. Для школьницы это самый топ :)
