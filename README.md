# Проект: Очистка воды (7 класс)

Сделали с сестрой Яной (7 «А» Курчатовского класса) годную презентацию по биологии+информатике
Тут не просто слайды, а интерактив на стыке экологии и IT.
Она потихонько вливается в программирование

### Суть

- **Тема:** Технологии очистки воды (от Оки до СанПиНа).
- **Междисциплинарность:** Биология (активный ил) + Химия (коагуляция) + Информатика (SCADA).
- **Стек:** React, Framer Motion (для плавных карточек), Tailwind/HeroUI.

### Фишка

Вместо обычного PowerPoint запилили веб приложение с анимациями. Для Курчатовского класса это самый топ :)
